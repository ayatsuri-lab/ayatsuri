# @ayatsuri-lab/ayatsuri-openbsd-x64

Platform-specific binary package for Ayatsuri.

This package contains the pre-compiled Ayatsuri binary for OpenBSD x64.

## Installation

**Do not install this package directly!**

This package is automatically installed as an optional dependency when you install the main package:

```bash
npm install @ayatsuri-lab/ayatsuri
```

## About

This is a platform-specific distribution of [Ayatsuri](https://github.com/ayatsuri-lab/ayatsuri), a powerful workflow orchestration engine.

For documentation and usage, please see the main package: [@ayatsuri-lab/ayatsuri](https://www.npmjs.com/package/@ayatsuri-lab/ayatsuri)
